# AI Safety Incident Dashboard (TypeScript)

## Setup Instructions

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm start
```

## Tech Stack
- React (TypeScript)

## Features
- Display AI safety incidents
- Filter by severity
- Sort by report date
- View incident details
- Report new incidents
